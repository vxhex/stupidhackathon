About

Stupid Hackathon Chicago is a one-day event where designers, developers, and dreamers join hands to build useless things. We're the Chicago cousins of NYC's Stupid Hackathon.

We are building a web 3.0 synergy of entrepreneurs. You must install NodeJS to use this website.

Organized by vxhex, superfamousguy, and chicohernando.
